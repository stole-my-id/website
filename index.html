stole-my.id

stole-my.id is a silly subdomain service, allowing you to switch from boring domains from hosting providers like pages.dev, vercel.app, github.io and etc.

To register a stole-my.id domain, please read the README in our repository.

How does this work?

stole-my.id is powered by DNSControl and Cloudflare, which allows DNS records to be made via our Github repository.

As of February 13, 2026, stole-my.id supports many types of records, such as A, AAAA, CNAME, MX, TXT, NS, SRV, and DS
